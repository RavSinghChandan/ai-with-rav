## How this folder works
- Each video's content lives in `days/dayNN.md` (title + content only).
- Run the factory to build the branded PDF:
  ```
  python3 ../../build_master_pdf.py topics/11-system-design/days/dayNN.md
  ```
- The cover (logo + photo + "System Design" + DAY badge) is automatic — identical style across all topics.
